Especialidades Front-End

Figma

Herramienta para desarrollo de prototipos. Podrás ver ejemplos del proyecto con fuentes, colores y estructuras definidas.

HTML

Lenguaje base estructural para páginas web.

CSS

Lenguaje de diseño. Aportará estilo a la estructura de HTML, como por ejemplo, colores, distribución, bordes, fuentes y similares.

Bootstrap

Es un framework de código abierto multiplataforma para diseños web adaptables.

Javascript

Es un lenguaje de programación que añade características interactivas, por ejemplo, animaciones.

Ruby

Si bien es un lenguaje de programación Back-End, utilizado en desarrollo de aplicaciones web, si el proyecto lo precisa, recurriremos a Ruby.

Sass

Preprocesador de CSS, que permite generar multiples funciones de guía y definición, extremadamente útiles en proyectos de gran magnitud.

Git

Es un software de control de versiones, pensado para la eficiencia y mantenimiento de versiones cuando muchos archivos de código.

GitHub

Sistema de control de versiones que permite almacenar repositorios de archivos de código de manera online.
